# TIL

## 학습한 내용을 꾸준히 기록하고 이후에 참고하기 위한 레포지토리:

[김남준](https://github.com/namjunemy/TIL)님의 TIL을 참고하여 만들어보려 한다.

## 작성 규칙

- 주제별로 분류하여 기록한다.
- 꾸준한 학습을 지향하되 기계적인 일일커밋을 지양하기 위해 날짜는 기록하지 않는다.
- 폴더와 파일명은 영문으로 작성한다.
- Reference를 명시한다.
